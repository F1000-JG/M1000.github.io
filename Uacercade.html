Aprende paso a paso

Aquí encontrarás ejemplos básicos de señas para comunicarte mejor. Puedes usar imágenes, descripciones y practicar día a día.

Letra A

Cierra el puño con el pulgar hacia un lado. Es la seña para la letra A en el alfabeto manual.

Letra B

Abre la palma con los dedos juntos hacia arriba y el pulgar doblado sobre la palma.

Letra C

Mantén la mano como si sostuvieras una taza, formando la letra C con los dedos.

Letra D

Levanta el dedo índice y mantén los demás dedos en un círculo para formar la letra D.
